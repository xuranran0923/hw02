## 任务二：DeepSeek大模型调用Chatbot示例代码
### 一、开发环境与依赖
1. 运行环境：Python 3.8及以上版本
2. 依赖库：openai
3. 安装命令：pip install openai

### 二、调用平台与配置信息
1. 调用平台：火山引擎方舟平台（Volcengine Ark）
2. 接口地址：https://ark.cn-beijing.volces.com/api/v3
3. 已配置信息：代码中已填入火山引擎API Key及DeepSeek模型推理接入点ID（ep-20260318094713-5rh82）

### 三、代码运行方式
1. 将代码文件保存为deepseek_demo1.py；
2. 打开终端/命令行，进入代码所在目录；
3. 执行运行命令：python deepseek_demo1.py；
4. 程序将自动调用DeepSeek大模型，输出问题的推理过程（若有）及最终回答。

### 四、问答示例
1. 提问内容：脑电检测仪的作用是什么？
2. 输出形式：先打印模型推理过程（DeepSeek模型特有功能），再打印最终答案，无推理过程时直接输出答案。

### 五、代码功能说明
实现**非流式单次问答**功能，通过OpenAI兼容接口调用火山引擎上的DeepSeek大模型，支持获取模型推理过程与核心回答，代码结构简洁，贴合大模型API调用基础作业要求。
